# Face Recognition

A face recoginition system with life proof verification using cloud points.

# Getting Started
## Prerequisites
```
sudo apt install cmake
sudo apt install g++
sudo apt install python3-dev
```

```
pip install -r requirements.txt
```

## Run
```
python main.py
```
